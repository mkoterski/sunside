Map tiles load from OpenStreetMap — needs a connection. The route line is colored by shaded side, same as the list below.
How this works. Every leg between two stops has a compass bearing. Given the sun's position for the chosen time, each leg is scored by which side it strikes — the opposite side is the shaded seat. Legs are weighted by length, so the verdict reflects where you spend the ride, not just the count of segments. Coordinates are real stop locations; the live app reads them from VBB per trip.

Note on the stop list. This is an approximate sequence assembled from public route maps, which disagree on a stop or two. The full line runs ~32 stops end to end; this trip is a subset of it. The authoritative per-trip stop list and exact coordinates come from VBB's live trip data, which the real app uses.

Demo location: Berlin (52.55°N, 13.40°E). Sun position computed inline (NOAA algorithm) — no external dependencies.
