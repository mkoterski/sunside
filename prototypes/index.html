SunSidePROTOTYPES

Self-contained, offline prototypes of SunSide ideas and flows. Each opens directly in the browser — no server, no API, no build. They share the verified sun-position maths and per-segment shade logic. The deployed app lives in public/index.html; these are exploratory builds.

010
Shade demo →
Tram 50 · Hugenottenplatz → U Seestraße
Offline demo of the segment-aware shade logic on one real trip — the verdict, a shade meter, and a stop-by-stop breakdown with a time slider.
011
Full journey →
Tram 50 · Hugenottenplatz → U Seestraße
Clickable prototype of the whole user journey: stops near you → pick line → destination → where to sit.
012
Auto-play ride →
Tram 50 · Hugenottenplatz → U Seestraße
A second take on the "follow the ride" screen — the journey view auto-advances on a timer, with play/pause and auto-scroll.
013
Regional rail →
Bus 154 → S2 → RE3 · Aubertstraße → Ferdinandshof
Long-distance, multi-modal case that leaves the Berlin network and runs on regional rail toward Stralsund.
014
City MetroBus →
M21 / X21 · Wedding → Goerdelersteg
A city bus run where the shaded side flips on the final northeast hook into the stop. Stops and bearings are real, from OpenStreetMap.

Numbered in the order they were built. Each folder has its own README with the details; see prototypes/README.md for the index.
